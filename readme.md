# Unreal Engine 5 Gameplay Ability System Crash Course

## Overview
A comprehensive crash course on the Gameplay Ability System (GAS) in Unreal Engine 5, from Stephen Ulibarri in which we build a fully-functional GAS compatible game from scratch using both C++ and Blueprints.

Course Link: [Unreal Engine 5 Gameplay Ability System Crash Course](https://www.udemy.com/course/unreal-engine-5-gameplay-ability-system-crash-course/)

## Topics Covered

### Core Systems
- Enhanced Input setup with separate Input Mapping Contexts.
- Custom Ability System Component functionality.
- Gameplay Tags creation in editor and C++.
- Gameplay Cues for animations, particles, and camera effects.

### Abilities & Actions
- Multiple ability types: attacks, hit reacts, AI behaviors.
- Ability Tasks for gameplay events and montages.
- Custom Latent Task nodes in C++.
- Ability costs and cooldowns.

### Game Mechanics
- Health and Mana pickups.
- Gameplay Effects for attributes and tags.
- UI integration with progress bars and damage numbers.
- Niagara particle systems on the GPU.
- Hit detection and collision optimization.
- Character death and respawning.
- Enemy AI searching, chasing, and attacking.

### Development Skills
- Blueprint to C++ conversion.
- Custom Animation Notifies and Notify States.
- Overlap tests and collision filtering.

## Project Foundation
- Start with a blank Unreal Engine 5 project.
- Import free Paragon assets from Epic Games.
- Build everything from the ground up using GAS.
- Mix of C++ and Blueprints for flexibility and performance.
